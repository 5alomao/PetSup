<h1 align="center">PetSup</h1>
<h4 align='center'>Sistemas desenvolvido em C# conectado com Banco de Dados !</h2>

### Linguagens Utilizadas:
* C#
* SQL

### Ferramentas Utilizadas:
* Microsoft Visual Studio
* MySql Workbench
* HeidiSQL
* Xampp

### Funções do Software:
* Inserir dados;
* Listar dados;
* Alterar dados;
* Remover dados;
